Known Issues



SoarTech Bug #1404: External modifications to agent not noticed by open agent editor

If an agent is modified externally to the agent editor, say through the links at the top of the Soar file editor, the open agent editor does not notice the changes.

When changes are made externally, the agent editor should notice and pop up the standard "Reload changes from disk?" dialog box.

SoarTech Bug #635: Productions not detected or parsed

Productions which are created completely through Tcl calls are not detected or parsed by the Soar IDE.
